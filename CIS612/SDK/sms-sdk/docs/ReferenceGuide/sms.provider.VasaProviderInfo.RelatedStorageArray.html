Data Object - RelatedStorageArray

Property of
VasaProviderInfo
Extends
DynamicData
Since
SMS API 4.0

Data Object Description

Indicates whether the provider has been marked as active for the given array for the session context. SMS uses priority value to mark a provider as active among the ones that are registered with SMS and manage this array.

Properties

Name Type Description
activexsd:boolean

This field indicates whether the provider is currently serving data for the StorageArray
arrayIdxsd:string

uuid of StorageArray
manageablexsd:boolean

Manageability status of StorageArray on VASA provider, if true it is manageable.
priorityxsd:int

priority of StorageArray For VASA 1.0 providers, this field is set to -1.
