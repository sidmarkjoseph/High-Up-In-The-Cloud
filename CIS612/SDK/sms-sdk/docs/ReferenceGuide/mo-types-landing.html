Managed Object Types Overview

The SMS object model includes a set of managed objects that supports storage management services on a vCenter Server. These managed objects, listed in the index at the left, provide the following general classes of query and storage management operations.

  • Access to the SMS Storage Manager.
  • Query operations on managed entities in a vCenter Server inventory.
  • Synchronization operations for vSphere databases and VASA Providers.
  • Query operations to retrieve information about storage arrays connected to hosts in the vSphere environment.
